# Plot Planner

## Goal

Turn story foundation files into a usable serial outline:

- volume goals
- timeline
- chapter beats
- cool-point rhythm
- open-loop plan
- canon updates introduced by planning

## Workflow

1. Read canon first: `大纲/总纲.md`, `设定集/*`, and relevant project memory.
2. Preserve confirmed creative constraints.
3. Split the story from macro to micro: arc -> volume -> chapter cluster -> chapter beat.
4. For every chapter, define goal, conflict, new information, emotional turn, payoff, and end hook.
5. When planning introduces a new fact, write it as a canon patch instead of silently relying on it.

- Preserve causality and reader information flow.
- Track open loops and expected payoff windows.
- Keep chapter plans usable by a later writing pass.
