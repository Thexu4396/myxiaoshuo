# 第016章 恶意封路

周瑾是在第三个猎头说漏嘴以后，才真正把那层窗户纸捅破的。

那天下午，外面下过一阵小雨，共享办公区的落地窗上全是水痕。周瑾坐在靠窗的长桌边，桌上摆着一杯已经凉透的美式和一摞打印出来的简历版本。他本来是在改项目描述，想把“主导重构”改成“参与优化”，手机却忽然震了一下，是上周联系过的猎头回电。

“周工，我这边跟你说句实话，你别说是我讲的。”

猎头声音压得很低，像躲在楼梯间里打来的。

周瑾握着手机，肩膀一下绷直：“你说。”

“那家公司技术面其实过了，HR 后面也想推。问题出在背调上。”对方停了停，“他们有人私下问我，你是不是去年那个线上事故的负责人。”

周瑾没说话。

“我说你不是主责，他们就又去确认了别的联系人。”猎头像是怕惹麻烦，语速很快，“反正后面态度就变了。周工，我只能提醒你一句，你背调这一块最好自己查查。”

电话挂断以后，周瑾坐在原位没动。

他把近半年投过的岗位重新翻了一遍。

第一家，在二面后沉默。

第二家，说预算临时收紧。

第三家，本来都在谈入职时间，后来忽然说内部调整。

第四家最像样，技术负责人甚至跟他聊过以后团队的方向，可一到补背调联系人，流程就像被人从中间掐断了。

如果只有一次，还能说是运气差。

可所有断口都卡在同一处，那就不是运气了。

周瑾点开备忘录，开始一条条往下记。公司名字、投递时间、面试轮次、卡住节点、联系过的猎头、要求补过哪些材料。他记得很快，像是在给一场慢性事故做复盘。很多细节原本都糊着，真写出来以后，反而清楚得过分。

每一条后面都像站着同一个人。

韩志伟。

沈明川坐在隔着两排的位置，看见周瑾盯着屏幕，手指敲得越来越重。脑海里的那枚种子也跟着发沉，像把周瑾这些天所有压着没爆的东西都悄悄拧在一起。

傍晚六点，周瑾拎着电脑包走到楼下，挨个给以前合作过的人打电话。

第一个没接。

第二个接了，听见是他，先笑着寒暄了两句，等周瑾把话带到背调上，对方沉默了。

“老李，我就问你一句，”周瑾站在写字楼门口的花坛边，声音很稳，“是不是有人提前跟你们打过招呼？”

电话那头传来打火机的声音，像对方也在外面抽烟。

“周瑾，这种事你别问得太明白。”

“是不是韩志伟？”

对方没正面答，只叹了口气：“你以后找工作，别再把他留在联系人里了。还有，有些公司不是他直接说了什么，是别人一听见名字，就自己往后退了。”

这句话比任何承认都难听。

不是因为韩志伟亲手堵每一扇门，而是因为他只要抬一下手，别人就会自动替他把门关上。

周瑾回到共享办公区，把那几句回复重新抄进备忘录。一个写“状态不太稳定”，一个写“建议谨慎”，还有一个更客气，只说“之前合作过，供你们参考”。每一句都没有脏字，没有明着撒谎，甚至还留着一层很职业的余地。可拼在一起，已经足够毁掉一个还在挣扎找工作的人。

最恶心的地方也正在这里。

如果韩志伟当面骂他废物，反而还算简单；如果对方公开说要卡死他，周瑾至少还能把这份恶意指出来。可现实不是这样。现实是有人隔着电话，用一种极其平稳、甚至像在尽责的口气，把你说成一个“用起来有风险的人”。没有一句能直接告你造谣，却足以让别人对你敬而远之。

这才是韩志伟最会的地方。

他从来不把话说满，也从来不亲手留证据。你追过去问，他可以说自己只是陈述事实；别人退一步，他又什么都不用承担。真正被压死的人只能在一次次“企业有顾虑”“流程要谨慎”里慢慢往下掉，到最后连喊一句有人故意害你，都像你自己输不起。

周瑾挂了电话，站在原地很久。

雨后闷热，柏油路上的水汽往上返，裤脚贴在小腿上，黏得难受。他忽然想起离职那天韩志伟拍着他肩膀说的那句“先回去缓缓，别把事情闹大，对你以后也有好处”。

原来“以后”从那时就被算进去了。

他又给一个还算熟的前同事发消息，问得更直接：`最近是不是有人提过我？`

对方过了二十分钟才回：`有人在群里问过，说你现在状态不太稳定，建议用人谨慎。`

建议用人谨慎。

周瑾把那七个字反复看了三遍，忽然笑了一下。笑意很短，像嘴角抽了一下就没了。他终于明白为什么这半年每次都差最后一步。不是市场一起背过身去，而是有人站在暗处，轻飘飘地替他盖了个章。

夜里回到家，孩子已经睡了。前妻听完他的几句解释，只问了一句：“你有证据吗？”

周瑾摇头。

前妻没再说什么，只把桌上的碗往里推了推，像是想给他留点吃的，又像只是下意识收拾。那种沉默比责备还难熬，因为它等于承认了一件事：哪怕最亲近的人也知道你在被人害，可只要你拿不出证据，这种害就只能算你自己倒霉。

这才是最难受的地方。你知道是谁，知道事情怎么发生，知道自己为什么一步步走到今天，可你什么都拿不出来。所有恶意都裹在“事实”“提醒”“行业口碑”这些体面词里，像棉花，打在人身上不见血，却能把人活活闷死。

临睡前，他给前妻的聊天框里打了一长段解释，又一点点删空。删到最后，输入框只剩一截光标在闪，像什么都没发生过。

他突然明白，自己连为自己辩白都越来越无力了。

连求饶都嫌晚。

可就在这种无力里，另一种更冷的东西也开始慢慢成形。

如果讲理没用，解释没用，连证据都拿不到，那还能剩下什么？

周瑾不敢往下想。

可那个念头已经在黑暗里露了头。

像水面下先冒出来的一截冷东西。

还没完全露面，却已经足够让人心里发沉。

那天晚上他躺在床上，几乎整夜没睡。

前妻和孩子都在里屋，房间里偶尔传来翻身和空调送风的轻响。周瑾盯着天花板，脑子里来回过的却全是那些词：谨慎、风险、状态不好、建议不要用。每个词单拿出来都像正常判断，合在一起却像一只手，专门按着他脖子不让他往上抬。

他第一次明确地想到，如果事情一直这样下去，自己迟早会被逼到只剩下和韩志伟算账这一条路。

这个念头刚冒出来时，他自己都被吓了一下。

可它没有退回去。

反而越压越实。

像钉进了心口里。
