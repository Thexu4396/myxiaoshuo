# 第039章 去监狱前一晚

判决下来以后，时间忽然就变快了。

以前周瑾待在看守所里，日子是一格一格熬。今天和明天没多大区别，最多只是律师哪天来、提讯哪天排、外面天气变没变。可判下来之后，很多事情第一次有了明确方向。什么时候办移交，什么时候走手续，什么时候把人从这里送去监狱服刑，都开始变成一句句说得出日期的话。

模糊的等待被替换成倒计时。

这反而让人更清醒。

监室里有人听说他要走，语气里还带点羡慕。

“至少算定了。”

“进去熬几年，出来就完了。”

“总比这么吊着强。”

周瑾听着，没接话。

只有真正坐在这个位置上的人才知道，“定了”未必是松气。对于很多人来说，刑期落下来，后面的路就只是熬。可对周瑾而言，这几天反而像某个窗口在一点点关上。看守所里他已经摸熟了节奏，摸熟了人，摸熟了自己那股力量能往哪儿走、什么时候试最稳。真到了监狱，一切都会被重新打散。

如果要走。

就不能等到那时候。

这个判断在判决当天晚上还是一团模糊的影子，到第二天，已经慢慢长成了一根很直的线。周瑾开始观察得更仔细。押解的人通常几个。手铐什么时候上。上车前会不会短暂停顿。下车时人和人之间会不会为了脚步快慢出现一瞬的错位。很多东西他以前不是没见过，只是没把它们和“出去”这件事真正连起来。

现在不一样了。

他不需要一整套完美到滴水不漏的计划。

他只需要一个够短、够乱、够让人来不及反应的窗口。

这一点，和那晚韩志伟扑上来时，其实没什么两样。

都是只吃一瞬。

家属那边没有再来。

律师倒是又见了他一次，把后续申诉、减刑、表现这些正常服刑里会用到的词都讲了一遍。语气还是平稳的，像在给一个已经进入确定轨道的人解释接下来每个站点会是什么。周瑾听着，也点头，偶尔应一句“明白”。

他表现得很配合。

配合到连律师都像稍微松了口气。

“现在最重要的，就是别再折腾。”临走前，对方说，“四年不是不能熬。你前面那些冲动，到这一步也该收了。”

周瑾隔着玻璃看着他，神情平得几乎没有起伏：“嗯。”

律师离开以后，他却在心里很轻地想，来不及了。

不是冲动来不及收。

是整个人已经不在原来那条轨道上了。

夜里，监室很安静。

窗外没什么风，只有远处偶尔传来的模糊车声。周瑾躺着没睡，手腕慢慢贴上床沿，感受皮肤下面那股越来越熟悉的沉意。它已经不像最开始那样一碰就乱，也不像中间那样总带着压不住的蛮劲。现在更多的时候，它是静的。像一条线伏着，等你把注意力送过去，才会跟着往里收。

他闭上眼，在脑子里一遍遍过手铐的结构。

不是凭空想象。

是那天押回来看见、碰到、感受到的实物印象被反复拆开。哪一截最窄，哪一截连接着铰链，哪一处表面看着厚，其实内部受力最集中。周瑾甚至开始用程序员的习惯去理解它。不是“这东西多结实”，而是“它最依赖哪一点完整”。只要那一点先崩，剩下的强度都会立刻变样。

想到后来，他心跳都没怎么快。

只是越来越稳。

稳得近乎冷。

旁边床上有人睡梦里翻身，铁架床轻轻一响。周瑾睁开眼，视线停在头顶那块昏白灯影上，忽然觉得自己这两个月真正练出来的，不只是寸崩。

还有另一种东西。

忍。

等。

把所有多余情绪压下去，只留下那个最有用的点。

从前的周瑾不是这样的人。以前他就算被逼急了，心底还是总有一点想把话说清、把理掰明白的习惯。现在那部分还在不在，他自己都说不准了。至少这一刻，他脑子里最清楚的不是对错，也不是公道，而是下一次上铐时，自己应该把力落在哪儿。

外面的流程还在照常走。

有人会在文件上签字。

有人会核对移交名单。

有人会在押解前例行检查。

一切都会按规矩来，严丝合缝，像这个系统永远不会给人留下空子。周瑾却知道，再稳的流程，最后也得落在人身上。人会累，会分神，会凭经验觉得一切都已经定了。只要其中一瞬稍微松一点，他就有机会把那道缝撬出来。

想到这里，他慢慢把手收回来，掌心贴在胸口。

胸腔起伏平稳，耳边没有轰鸣，头也没发胀。那股力量像终于和他达成了某种默契，不再只是危险地寄生在体内，而开始像他身体里的一部分。

第二天什么时候走，还没人明确告诉他。

可周瑾知道，就快了。

而他也已经准备得差不多了。
