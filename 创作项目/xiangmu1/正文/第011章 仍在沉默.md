# 第011章 仍在沉默

周瑾拿着那家公司的回复，在楼下站了很久。

太阳已经偏西，玻璃幕墙把天色照得发白。他没有立刻走，也没有打电话追问为什么，只是把手机举起来又放下。屏幕一亮，上面还是那句客客气气的“岗位暂时有变动，后面有机会再联系”，一点毛病都挑不出来。

他站了十几分钟，才慢慢往共享办公区走。

这几天，沈明川已经习惯了这种隐秘的跟随。不是刻意盯梢，只是会下意识去留意周瑾在哪儿，什么时候低头，什么时候停步，什么时候又把那副勉强还算体面的壳重新穿上。周瑾没有察觉，沈明川也不敢贸然靠近。他只知道，那枚被自己投进去的“东西”还在。它没长出来，没发光，没给现实留下一点超自然的痕迹，却像已经在周瑾身体里安了根。

共享办公区前台的小姑娘照例把工位卡递给周瑾。

“今天还是老位置。”

“嗯。”周瑾接过去，勉强笑了笑。

工位边上坐着两个做销售的年轻人，正在打电话拉客户，一个声音很亮，一个嗓门很大。再过去一点，有个做跨境电商的女生开着视频会，耳机里英文和中文来回切。周瑾在这群人中间坐下，把电脑打开，先盯着桌面发了会儿呆，才开始整理简历。

他删掉了“负责”两个字，又删掉了“主导”两个字，最后只留下“参与”。连那些最基本的项目描述，他都改得尽量保守。以前他总觉得，做过什么就该写什么，现在才知道，写得越真，别人越容易顺着某个词把你往下压。

旁边那两个销售还在打电话。

“姐，我们这个方案真的适合您。”

“没关系，您再考虑也行。”

“价格不是问题，关键是先合作起来。”

周瑾听着这些话，手指停在键盘上，有一瞬间甚至想笑。原来所有行业里，求人给机会的时候，话术都差不多。只不过别人是求客户，他现在是求别人别一眼就把自己筛掉。

晚上八点多，他终于把简历改完，重新投出去三家。

发完以后，他靠在椅背上，盯着“已投递”的提示发呆。屏幕光落在脸上，把眼底那点疲色照得很清楚。过了会儿，他低声说了一句：

“总会有一家公司要人的。”

声音很轻，像是怕说重了，连这句话都会破。

他说完以后，又把刚投的三家岗位重新点开，逐条看任职要求，生怕自己漏了什么。哪怕其中一家的工作地点远到单程要一个半小时，他也在备忘录里记下了面试时可能会被问到的问题。像只要准备得足够细，结果就能往自己这边偏一点。

连通勤方案都顺手记了一版。

中途前妻给他发了条消息，问孩子下周的培训费能不能先转一半。周瑾盯着那条消息看了几秒，回的是：`明天我想办法。`

发完以后，他把聊天框往上划，最上面还是自己前一天发的那句“可能有戏”。对方没接那句，只接了钱的事。周瑾看着那一来一回，脸上一点表情都没有，只是把手机扣过去，继续改简历。像一个人如果不停手，很多事情就还能先不去想。

过了会儿，共享办公区开始清场。保洁阿姨推着车从过道慢慢走过去，提醒还没走的人把垃圾带一下。那两个做销售的年轻人终于打完电话，边收东西边抱怨客户难缠。周瑾还坐在原位，像没听见，直到前台来提醒十二点要断电，他才“嗯”了一声，把电脑先合上一半。

沈明川就坐在隔壁桌，手里拿着笔，假装在写东西，心里却越来越不安。那枚种子沉得比前几天更明显了，像一枚压在水底的钉子。他能感觉到它在周瑾身上找地方扎根，却仍然看不见现实里有半点异样。

这种沉默比爆发更难熬。

没有结果，也没有征兆。

好像一切都跟平时一样，可正因为太像平时，才让人发慌。

十一点刚过，周瑾接到一个猎头电话。

对方说有个岗位可以试试，做企业级中台，薪资不算高，但至少是正经团队，不是外包，也不是临时项目。周瑾本来已经准备关电脑了，听完这几句，整个人都坐直了一点。

“具体技术栈是 Java 吗？”

“上班地点在哪边？”

“面试什么时候可以安排？”

他问得很快，却还是努力让语气听上去不那么急。猎头那边大概也能听出他的紧绷，说话难得耐心，甚至还安慰了一句：“这家公司招得急，你条件也匹配，说不定很快就能定。”

挂电话以后，周瑾盯着手机看了好一会儿。

然后把笔记本重新翻到第一页，拿笔把第二天要准备的项目重点又写了一遍。

他写得很认真，连每一段停顿都标出来了。技术亮点写在左边，事故解释写在右边，最后还单独圈了一句：`不要急，不要先提上家公司。`

这种细到近乎笨拙的准备，让沈明川看得更难受。周瑾不是没在努力，他只是所有努力都像落进了一个早就被人动过手脚的漏斗里，怎么倒都装不满。

临走前，周瑾还去前台借了充电器，说手机电量太低，怕明早猎头联系不上。前台小姑娘把线递给他时笑着说：“你最近业务挺忙啊。”周瑾也跟着笑了一下，说“是，有点赶”。那笑挂在脸上不过两秒，转身时就掉了。

回去的路上，他在地铁里都没坐下，一只手抓着吊环，另一只手还在翻备忘录。项目经历、离职说法、薪资底线、通勤时间，能准备的都写了。像只要把这些东西一遍遍过熟，明天就真能稳一点。

可沈明川知道，真正决定结果的，也许根本不是这些。

沈明川看着他低头写字的样子，忽然生出一种很荒唐的感觉。

一个人只要给他一点点希望，他就又会立刻坐直，立刻开始准备，立刻告诉自己明天说不定会更好。可偏偏就是这种“说不定”，一次次把人吊住，再一次次摔回去。

周瑾当然不知道，电话挂断以后，另一头的流程已经开始走了。

简历会被转给 HR。

HR 会按惯例做背景调查。

而那家“愿意试试”的公司，名单上也会很快出现韩志伟的号码。

夜里回到出租屋后，沈明川又一次进了本源空间。

灰雾里的那枚种子没有亮，也没有再生出别的异样，只是比白天看上去更沉了一点，像有人在它表面又压了一层土。沈明川蹲在那儿盯了很久，忽然有种很坏的直觉。越安静的东西，往往越不是在消失，而是在等。

可他什么也做不了。

他甚至连提醒周瑾一句“别再往前走了”都不知道该怎么开口。

因为他自己，才是最先推了那一下的人。

这一点，连他自己都开始怕了。
